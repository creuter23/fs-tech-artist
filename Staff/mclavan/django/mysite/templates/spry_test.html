35% - Project - Prop Rig
Prop rigs are a great place to start learning how to rig. They give a quick testing ground to student different types of rigging techniques, while finshing off a rig with out too much time invested.
Level 0 - Placement Rig
Placement rigs are required for all rig, where it be character or prop rig. This allows the layout artist to put the rig in its starting place. It also hold all the layer system for the rig as well.
Level 1 - Control Icons and Constraints
A Level 1 props have to have some of control icon and constraint system. While this prop doesn't require a bind, the geometry should be grouped into its moving parts and have its pivot points set properly.
Level 2 - Bound Prop
This prop requires a bound serface along with a control system to move the joints. This rig can also include IK (Inverse Kinematic) systems.
Level 3 - Complex Systems
These props take advantage or direct connections and set driven keys. Other types of systems are acceptable like rigged textures and the visual systems that they represent.
35% - Project - Tools
Different tools will require differnet problem solving techniques to complete. In hense there are four levels of tools to tackle.
10% - Toolset Project
Organize and collect all the tools built during the month and place them into a overall toolset. New tools don't have to be created here. Just bring everything together and nice logical order.
10% - Character Rig
The Character rig pulls from all the development you've done on the props.
